Remarks

This method uses reflection to serialize the public properties in the config.

A property will not be persisted if,

  • The value is equal to the default value, or
  • The string representation of the value is an empty string, or
  • The property has a Browsable(false) attribute, or
  • The property has a NonPersisted attribute.
